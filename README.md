# Getting Started

### Install from CRAN

```R
install.packages("shinyFeeedback")
```

### Or Install Dev Version from GitHub

```R
devtools::install_github("merlinoa/shinyFeedback", build_vignettes = TRUE)
```
### Read Intro Vignette

```R
# load package
library(shinyFeedback)
# read intro vignette
browseVignettes(package = "shinyFeedback")
```
